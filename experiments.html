FITIX

PC · Clon · Game Maker
Finalizado
Experimento

FITIX es un clon experimental de Tetris desarrollado como ejercicio de lógica, control y ritmo de juego. El objetivo fue recrear las mecánicas clásicas del género, afinando la caída de piezas, la detección de líneas y la respuesta del jugador, todo con una estética simple y retro.

Prototipo #01 - Walk Sim

PC · RPG/Walk Sim · Game Maker
Experimento

Prototipo enfocado en la exploración y la construcción de un mapa de gran tamaño. Este experimento prueba un sistema de música dinámica que cambia según la zona, además de la implementación y testeo de eventos básicos dentro del entorno.

Prototipo #02 - RPG

PC · RPG · Game Maker
Experimento

Un RPG simple creado para experimentar con sistemas de diálogo. El enfoque principal está en la interacción con personajes, la presentación de textos y la estructura de conversaciones, sirviendo como base para futuras mecánicas narrativas.

Prototipo #03 - Plataformero

PC · Plataformero · Godot
Experimento

Este experimento fue creado principalmente para aprender a utilizar el motor Godot y explorar las mecánicas básicas de un juego de plataformas 2D. Sirvió como banco de pruebas para implementar movimiento básico del personaje, detección de colisiones, diseño de niveles simples y la integración de elementos interactivos.
